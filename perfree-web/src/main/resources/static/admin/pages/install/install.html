PerfreeBlog

安装说明

欢迎使用PerfreeBlog,在开始前,您需要知道, PerfreeBlog同时支持 Mysql 和 sqlite 数据库, 我们默认推荐您使用mysql数据库带来更好的体验!

准备工作

一、 如选择Mysql数据库,则需要您提前安装好Mysql数据库,并提供以下信息:
1. 数据库用户名
2. 数据库密码
3. 数据库主机地址
4. 数据库端口号
二、 如选择Sqlite数据库,则不需要提供任何信息

如您已准备好以上工作,点击下方"下一步"按钮即可开始安装

许可协议
