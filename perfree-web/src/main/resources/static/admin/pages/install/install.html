Perfree-安装
欢迎使用perfreeBlog,在开始前,您需要知道, perfreeBlog同时支持 mysql 和 sqlite数据库, 我们默认推荐您使用mysql数据库带来更好的体验!

准备工作

1. 如选择mysql数据库,则需要您提前安装好mysql数据库,并提供以下信息:

  • 数据库用户名
  • 数据库密码
  • 数据库主机地址
  • 数据库端口号

2. 如选择sqlite数据库,则不需要提供任何信息

如您已准备好以上工作,点击下方"开始安装"按钮即可开始安装

许可协议
